# 🐍 Python Scripts & SysAdmin Notes

Добро пожаловать! Этот репозиторий содержит коллекцию моих учебных программ на Python и заметок по системному администрированию.

## 📁 Структура проекта

- **[python-basics/](./python-basics/)** — Простые скрипты для изучения синтаксиса (циклы, условия, функции).
- **[automation/](./automation/)** — Скрипты для автоматизации задач (резервное копирование, парсинг сайтов, отправка уведомлений).
- **[sysadmin/](./sysadmin/)** — Текстовые статьи, шпаргалки по командам Linux и примеры конфигов (Nginx, Apache, SSH).
- **[network-tools/](./network-tools/)** — Утилиты для проверки портов, пинга и анализа трафика.

## 🛠 Технологии

*   **Язык:** Python 3.9+
*   **OS:** Linux (Ubuntu), Windows
*   **Инструменты:** Git, Bash, Docker (в планах)
