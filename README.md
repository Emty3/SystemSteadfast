# 🐍 Python Scripts & SysAdmin Notes by Emty3

Добро пожаловать! Этот репозиторий содержит коллекцию моих учебных программ на Python, демонстрационные проекты и заметки по системному администрированию.

## 📁 Структура проекта

- **[Мои проекты/](./Мои%20проекты/)** — Мои основные проекты на Python: калькуляторы, игры, утилиты, боты и скрипты для автоматизации.
- **[Demo 3/](./Demo%203/)** — Демонстрационные примеры и тестовые скрипты для отработки новых технологий.
- **[sysadmin/](./sysadmin/)** — Заметки по администрированию: шпаргалки по командам Linux, конфиги веб-серверов (Nginx, Apache), настройка SSH и сетевых утилит.
- **[MySQL/](./MySQL/)** — Шпаргалки по SQL, скрипты для бэкапа/восстановления БД, конфиги `my.cnf`, оптимизация запросов.
- **[Ключи на Доктор веб/](./Ключи%20на%20Доктор%20веб/)** — Материалы и файлы, связанные с настройкой и управлением Dr.Web.

## 🛠 Технологии

*   **Язык:** Python 3.9+
*   **СУБД:** MySQL 8.0+
*   **OS:** Windows
*   **Инструменты:** Git, Bash, PowerShell, Командная строка, MySQL Client
*   **В планах:** Docker, CI/CD основы
